-> Descripción: Es un script en python, que manipula los csv para formatearlos de acuerdo al tipado que tenemos en base de datos.

-> Requisitos: En el directorio donde vamos a ejecutar el script debemos tener solamente los csv que vamos a tratar, el .exe y las instrucciones
   ¡¡ IMPORTANTE !! Asegurarse que no existen exportaciones pasadas que se hayan quedado en el directorio, en ese caso, tendriamos que eliminarlos.

-> Modo de empleo: Ejecutar el script con doble click, esperar 5/10 segundos para que cargue. En primer lugar solicitará la fecha de carga de los 
   registros ya que esta se hara manualmente y se imputará a todos los registros de la carga. Posteriormente tendremos distintas opciones para
   ejecutar una o las dos cargas juntas.

-> Si se modifica el numero, nombre u orden de las columnas, debemos revisar el código para adaptarlo al nuevo formato.
   los csv son abiertos gracias a las keywords ('activity' y 'payments') por lo que es importante no tener ningun fichero en el directorio
   que contenga dichas cadenas de caracteres.

-> Actualmente deja un csv preparado con el mismo nombre de columnas para importar en la base de datos de BI. El proyecto continuaría imputando directamente
  estos datos desde el script a la base de datos.
